# Release Notes: v0.1.4

Date: 2026-04-12

This release expands graph capabilities, improves LLM context controls, and strengthens query reranking.

## Highlights

- Added chunk-level and entity-level graph support in the core graph engine.
- Added graph export level control:
  - `--graph-level doc|chunk|entity`
- Added graph export formats:
  - `dot`
  - `json`
  - `cytoscape-html`
- Added chunk graph diagnostics:
  - `--show-chunk-graph-stats`

## LLM Context Improvements

- `--llm-context` now supports:
  - `--result-count <N>` with cap at `50`
  - `--llm-chunk-strategy all|by-doc`
  - `--simplified` (alias `--simplifed`) to keep envelope and return `top_chunks` as `string[]`
- Added chunk-level graph-aware scoring signals:
  - `chunk_link_score`
  - `entity_graph_score`

## Query Ranking Improvements

- Kept one unified hybrid ranker (no separate query strategy split).
- Added bounded graph-aware rerank features in existing score breakdown:
  - `graph_link_score`
  - `entity_graph_score`

## Ontology

- Added seed ontology export:
  - `--export-ontology`
  - `--ontology-out <path>`
- Ontology includes typed nodes/edges with confidence and provenance fields.

## Dependency Updates

- Added dev dependency:
  - `lru = "0.16.3"`
